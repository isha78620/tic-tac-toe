# 🎮 Tic Tac Toe Game

A simple and interactive **Tic Tac Toe (XO) game** built using **HTML, CSS, and JavaScript**.
This project demonstrates basic game logic, DOM manipulation, and event handling.

---

## 🚀 Features

* 🎯 Two-player game (O vs X)
* 🔄 Turn-based gameplay
* 🏆 Automatic winner detection
* 🔁 Reset & New Game functionality
* 💬 Winner announcement message
* 🎨 Clean and responsive UI

---

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (Vanilla JS)

## 🎮 How to Play

* The game is played between **two players**:

  * Player 1 → O
  * Player 2 → X

* Players take turns clicking on the grid.

* The first player to align **3 marks in a row (horizontal, vertical, or diagonal)** wins.

* Click **Reset** or **New Game** to start again.

---

## 🧠 Game Logic

* Uses an array of **winning patterns** to check results.
* After each move:

  * Checks if a player has won
  * Displays winner message
* Disables boxes after selection to prevent overwriting.
